# vp-cli-tools

![Npm 版本](https://img.shields.io/badge/we-cli_v0.0.1-green)

## 📖 简介

前端通用 cli 脚手架

## 模版

- ✅ [Vite4-Vue3-Typescript-template][https://kkgithub.com/github-learning/vue3-admin.git]

## 🏗 安装

```bash
npm install vp-cli-tools -g
```

## 📦 使用

```bash
vp-cli-tools create <project-name>
```

## 🚩 查看脚手架版本

```bash
vp-cli-tools -v

```
